
# SocketIO Quick Test
SocketIO Quick Test repo

## Prerequisites

 * git
 * Node + npm

## Download repo

```
git clone https://github.com/khai-test-repositories/socket.io-quick-test.git
```

## Installation

```bash
cd socket.io-quick-test
npm install
```

## Start server

```bash
npm start
```
